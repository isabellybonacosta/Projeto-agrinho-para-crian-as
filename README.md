# 🌱 Agricultura para os Baixinhos

Este projeto foi desenvolvido especialmente para o **Concurso Agrinho 2026**. O principal objetivo é apresentar uma plataforma web totalmente acessível, lúdica e divertida para que crianças possam aprender sobre o ciclo de vida das plantas e a extrema importância da atividade agrícola para o desenvolvimento do nosso país. 🇧🇷✨

---

## 🎮 Elementos e Interações do Site

A página foi projetada com foco na experiência infantil e conta com as seguintes ferramentas de engajamento:

* **🌱 Ciclo de Vida Dinâmico:** Um texto de apoio interativo e visual que explica o passo a passo do crescimento das plantas.
* **🕹️ Mini-Game "Faça Crescer":** Um jogo interativo onde as crianças vivenciam o papel do agricultor, aprendendo a ordem correta de preparar a terra, plantar, regar e dar sol para ver a plantinha dar frutos.
* **🎬 Cine Fazendinha:** Um espaço dedicado a vídeos instrutivos e animados para reforçar o aprendizado de forma lúdica.
* **🥛 Campo no Dia a Dia:** Exemplos práticos que conectam a atividade do campo com alimentos e produtos presentes na rotina dos baixinhos.
* **💡 Curiosidades Animadas:** Um sistema de perguntas e respostas estilo acordeão para sanar dúvidas divertidas sobre a agricultura.
* **♿ Acessibilidade Universal:** Painel com ferramentas de alto contraste e ajuste de tamanho de fonte, garantindo a inclusão de todos os usuários.

---

## 🛠️ Desenvolvimento Técnico

O ecossistema do site foi construído de forma modular e responsiva, utilizando o conceito *Mobile-First* para funcionar perfeitamente em computadores, tablets ou celulares.

As tecnologias aplicadas foram:
* **🌐 HTML5:** Estruturação semântica e tags de acessibilidade baseadas nas diretrizes ARIA.
* **🎨 CSS3:** Estilização viva, colorida e adaptável através de um *Design System* baseado em variáveis nativas.
* **⚡ JavaScript (ES6):** Lógica de estado para o funcionamento do mini-game, manipulação dinâmica do DOM e transições de tela.

O projeto foi inteiramente codificado através do editor **Visual Studio Code (VS Code)**, contando com o auxílio estratégico de Inteligência Artificial para a otimização, refatoração de código e garantia de boas práticas na arquitetura front-end.
